
# 欧亚成长引擎 (Eurasia Growth Engine)

> “愿这把火炬，能引来更多拾薪者。” — *致后来者*

## 🌟 项目概述

**Ez Hub**（全称：欧亚学生成长助手）是一款专为**西安欧亚学院人文教育学院学生成长促进会**（及其他学生组织）定制的现代化 Web 应用程序。它不仅仅是一个效率工具，更是一个集**任务管理、资源聚合、组织协作与身份认同**于一体的数字化平台。

本项目采用 **Local-First（本地优先）** 架构，无需后端服务器，所有数据存储于用户设备端，通过加密文件与彩色码技术实现数据的流转与同步。应用遵循 PWA 标准，提供媲美原生应用的流畅体验。

## 🚀 核心功能详解

### 1. 沉浸式任务中心 (Immersive Todo Center)
*   **极简美学**：采用流体渐变背景 (Liquid Background) 与高斯模糊 (Glassmorphism) UI 设计，配合呼吸光效与每日动态励志格言，提供极致的视觉体验。
*   **专注模式 (Focus Atmosphere)**：内置多种白噪音场景（听雨、篝火、森林、深海、深夜），支持可视化频谱跳动，并允许用户上传本地音频作为背景音。
*   **日历同步生态**：
    *   支持生成 `.ics` 文件并通过 Web Share API 分享至系统日历。
    *   提供 CalDAV 协议配置教程，帮助用户利用第三方服务（如 WPS 日历）实现双向同步与系统级强提醒。
*   **智能通知**：集成 Service Worker 与 Notification API，即使应用在后台运行也能通过本地推送提醒待办事项。

### 2. 多维工作台 (Workbench)
*   **动态部门架构**：
    *   默认内置七大核心部门（主席团、秘书处、生活部、学习部、文体部、志愿服务部、宣传部）。
    *   支持**自定义部门**创建，适用于不同学院或社团的组织架构。
*   **资源聚合卡片**：每个部门拥有独立的资源面板，支持添加、编辑、置顶、删除各类常用链接（如腾讯文档、表单、校内系统）。
*   **公告系统**：支持发布部门公告，具备置顶功能与“最新(NEW)”状态自动识别，确保重要信息传达。
*   **CIMBAR 离线同步**：
    *   独创的 **CIMBAR (Color Icon Matrix Barcode)** 技术，将部门配置编码为高密度彩色矩阵图。
    *   用户无需网络，仅需“扫一扫”管理员分享的彩色码，即可一键克隆部门的最新应用配置与公告。

### 3. 组织与身份系统 (Identity & Org)
*   **多组织切换**：支持用户创建或加入多个学生组织账号，并在不同环境间一键切换（数据完全隔离）。
*   **角色体系**：
    *   内置部员、干事、部长三级角色，支持自定义角色名称与英文头衔。
    *   注册流程通过粒子特效强化身份认同感。
*   **管理员权限体系**：
    *   **临时管理员**：有效期 12 小时。
    *   **长期管理员**：拥有完整的管理权限。
    *   **永久管理员 (Permanent Admin)**：最高权限拥有者，具备“无限传承”能力。
*   **权限传承 (Inheritance)**：
    *   通过 HMAC-SHA256 签名的加密二维码进行权限移交。
    *   **永久管理员特权**：作为开拓者，永久管理员在进行传承操作后**不会降级**，可多次授予他人长期管理员权限。
    *   **普通传承**：长期管理员在移交权限后需卸任。

### 4. 数据安全与隐私 (Security & Privacy)
*   **本地存储**：所有数据（个人档案、任务、配置）存储于浏览器 `IndexedDB/LocalStorage`，保障数据主权。
*   **AES-256 加密备份**：支持导出全量数据为 JSON 文件，并可设置 AES 加密密码，确保备份文件安全。
*   **灾难恢复**：具备故障恢复代码 (Failure Code) 机制，在权限激活失败时可生成加密凭证供人工审核恢复。

### 5. 交互体验
*   **全域触感反馈 (Haptics)**：集成 `navigator.vibrate`，在点击、切换、成功、警告等交互节点提供细腻的震动反馈（支持强度调节）。
*   **深色模式**：完美适配系统级深色模式，自动切换 UI 配色。

## 🛠 技术栈架构

本项目基于现代前端生态构建，追求极致的性能与体验：

### 核心框架
*   **React 18+**: 使用 Functional Components 与 Hooks 构建。
*   **TypeScript**: 强类型约束，确保代码健壮性。
*   **Vite**: 极速构建工具。

### UI 与 动画
*   **Tailwind CSS**: 原子化 CSS 引擎，实现复杂的玻璃拟态与响应式布局。
*   **Lucide React**: 现代化图标库。
*   **Canvas API**: 用于粒子特效（注册页背景）与 CIMBAR 彩色码的绘制/解析。
*   **CSS Keyframes**: 实现流体背景 (Blob Animation) 与 UI 进出场动画。

### 数据与逻辑
*   **Crypto-JS**: 核心安全库，用于 AES 数据加密、MD5 校验、SHA256/HMAC 签名验证。
*   **Service Worker**: 实现 PWA 离线访问能力与后台通知逻辑。
*   **Web Audio API**: 构建氛围音效引擎与可视化频谱。

### 独特实现
*   **CIMBAR Service**: 自研的彩色矩阵编解码服务，使用 8 色调色板 (3-bit) 进行数据编码，相比传统 QR 码拥有更高的数据密度和视觉辨识度。

## 📦 开发与部署

### 本地开发

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

### 构建

```bash
# 构建生产环境代码
npm run build

# 预览构建结果
npm run preview
```

### PWA 调试
由于 Service Worker 依赖 HTTPS 或 localhost，请确保在安全上下文环境中进行测试。
